# Bun API (Elysia)

High-performance Student Management API built with Bun and Elysia framework.

## Features

- **Elysia Framework**: Fast, type-safe web framework for Bun
- **Native Performance**: Leverages Bun's speed
- **Shared Logic**: Reuses packages from monorepo
- **Type Safety**: Full TypeScript support
- **JWT Auth**: Secure authentication
- **RBAC**: Role-based access control

## Getting Started

### Prerequisites

- Bun 1.2.x

### Installation

```bash
bun install
```

### Development

```bash
bun run dev
```

### Build

```bash
bun run build
```

### Production

```bash
bun run start
```

### Testing

```bash
bun test
```

### Benchmarking

```bash
bun run benchmark
```

## API Endpoints

Same as Express API - see main README for full documentation.

## Performance

Bun typically shows 2-3x better performance than Node.js for HTTP workloads.
